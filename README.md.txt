# Weather API Testing with Postman

This project contains a Postman collection and environment to test the live WeatherAPI.com API.

## Project Details

- **Collection:** `WeatherAPI.postman_collection.json`  
- **Environment:** `WeatherAPI.postman_environment.json`

## What this project demonstrates

- How to use Postman to test a real-world API with dynamic environment variables.
- Basic API request setup and parameterization.
- Using environment variables like `api_key` and `city` to make requests flexible.
- Exporting and sharing Postman collections and environments.

## How to Use

1. Import the collection (`collections/WeatherAPI.postman_collection.json`) into Postman.
2. Import the environment (`environments/WeatherAPI.postman_environment.json`) into Postman.
3. Select the imported environment.
4. Run the requests and observe the results.
