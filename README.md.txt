# 🌦️ Weather API - QA Testing Project (Postman)

This project demonstrates **API testing** using **Postman**, focused on fetching live weather data using [WeatherAPI.com](https://www.weatherapi.com/). It showcases the usage of Postman collections, environment variables, and assertions — perfect for showcasing API testing skills in QA roles.

---

## ✅ Features

- 🌐 Tested a live weather API with real-time responses
- ⚙️ Used environment variables to manage `API key` and `City`
- 🧪 Organized test cases into a Postman Collection
- 📂 Exported and version-controlled the Collection & Environment
- 📤 Pushed the complete project to GitHub using Git CLI
